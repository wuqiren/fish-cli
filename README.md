# fish-cli

一个 react 项目的脚手架
包含webpack 和 vite两种模板可供提供


# 如何使用



```
sudo npm install @tencent/fish-cli -g

mkdir 你的文件名(test)
cd test

fishcli create
按照你的需要输入选择即可

```


### 本地运行(自己想做修改的话)


```
git clone git@github.com:wuqiren/fish-cli.git
npm install

npm link
```


```
fishcli create 
```

按照自己的需要输入命令即可
